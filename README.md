
# Contacts Filter

A Contacts List filter created using html, materialize css and pure javascript.



## Run Locally

Clone the project

```bash
  git clone https: https://github.com/ankushbhagat124/Filterable-List.git
```

Go to the project directory

```bash
  cd my-project
```

Start the server

```bash
  Open Live Server
```
